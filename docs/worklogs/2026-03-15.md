## Key Learnings

- OCI ADB MongoDB hostnames must be read from resource outputs — never constructed manually.
- Terraform `null_resource` provisioners that depend on external tools should always validate tool presence or be replaced with HTTP-based alternatives (ORDS REST SQL works well here).
- ORDS schema enablement is a mandatory step for MongoDB wire protocol access — it is separate from user creation.
- Oracle does not support `GRANT CREATE INDEX`; index creation on owned tables is implicit.
- MongoDB URI database name must equal the Oracle schema/username on ADB.
- Always Free ADB instances require `lifecycle { ignore_changes = [compute_count] }` to avoid 403 errors on re-apply.
